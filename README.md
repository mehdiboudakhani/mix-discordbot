## üöÄ Installation
Clone the repository.
```bash
git clone https://github.com/mehdiboudakhani/mix-discordbot
```
Configure environment variables.
```bash
setx MIX_DISCORD_BOT_TOKEN "YOUR_DISCORD_BOT_TOKEN"
setx MIX_DISCORD_GUILD_IDENTIFIER "YOUR_DISCORD_GUILD_IDENTIFIER"
```
Launch the bot from the project folder.
```bash
dotnet run
```
